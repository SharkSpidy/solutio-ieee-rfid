# Public Assets

## QR Code Setup

Replace `gpay-qr.png` with your actual GPay QR code image.

The current file is a placeholder SVG. To use a real QR code:

1. Generate your GPay QR code
2. Save it as `gpay-qr.png` in this directory
3. The image will automatically appear on the payment page

## Favicon

Replace `favicon.ico` with your custom favicon if needed.
